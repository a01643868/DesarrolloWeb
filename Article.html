1. Variables en JavaScript

Las variables en JavaScript son contenedores para almacenar datos. Se pueden declarar utilizando las palabras clave let, const y var. A continuación, se detallan los aspectos fundamentales de las variables en JavaScript:

1.1 Declaración de Variables

Las variables se declaran utilizando las palabras clave let, const y var.

Por ejemplo:

  • const nombre = "Juan"
  • let edad = 30;
  • var salario = 3000

1.2 Tipos de Datos

1.2.1 Tipos Primitivos

Los tipos primitivos incluyen números, cadenas, booleanos, null y undefined.

1.2.2 Objetos

Los objetos son estructuras de datos complejas que pueden contener múltiples valores.

1.3 Alcance de las Variables

1.3.1 Variables Globales

Las variables globales son accesibles desde cualquier parte del código.

1.3.2 Variables Locales

Las variables locales solo son accesibles dentro de la función en la que se declaran.

1.3.3 Variables de Bloque

Las variables de bloque solo son accesibles dentro del bloque en el que se declaran (por ejemplo, dentro de un if o un bucle for).

1.4 Constantes

Las constantes se declaran utilizando la palabra clave const y no pueden ser reasignadas.

1.5 Diferencias entre let, const y var

  • let: Se utiliza para declarar variables que pueden ser reasignadas y tienen un alcance de bloque.
  • const: Se utiliza para declarar variables cuyo valor no cambiará durante la ejecución del programa y también tienen un alcance de bloque.
  • var: Fue la forma original de declarar variables en JavaScript, tiene un alcance de función o global y puede ser accesible fuera del bloque en el que se declara.

1.6 Uso de Variables en Práctica

Podemos utilizar variables para almacenar y manipular datos en aplicaciones web

Un buen ejemplo de esto puede ser crear una variable total en donde multiplicamos los valores que se tienen en precio y cantidad para luego imprimir esta variable en un console.log

2. Ciclos en JavaScript

Los ciclos en JavaScript permiten la repetición controlada de bloques de código. A continuación, se detallan los diferentes tipos de ciclos disponibles en JavaScript:

2.1 for Loop

2.1.1 Sintaxis y Funcionamiento

El bucle for en JavaScript se utiliza para repetir una porción de código varias veces.

Sintaxis: 

for (inicialización; condición; incremento) {
  // Código a ejecutar
}

Funcionamiento:

  • La inicialización se ejecuta una vez antes de que comience el bucle.
  • La condición es evaluada en cada iteración. Si la condición es verdadera, el bucle continúa. Si es falsa, el bucle se detiene.
  • El incremento se ejecuta después de que el bloque de código se haya ejecutado en cada iteración.

2.1.2 Ejemplos de Uso

Ejemplo de un bucle for que imprime los números del 1 al 5: 

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

2.2 while Loop

2.2.1 Sintaxis y Funcionamiento

El bucle while ejecuta una sentencia mientras una condición sea verdadera.

Sintaxis: 

while (condición) {
  // Código a ejecutar
}

Funcionamiento:

  • La condición se evalúa antes de que el bloque de código sea ejecutado.
  • Si la condición es verdadera, el bloque de código se ejecuta. Si es falsa, el bucle se detiene.

2.2.2 Ejemplos de Uso

Ejemplo de un bucle while que imprime los números del 1 al 5: 

let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}

2.3 do...while Loop

2.3.1 Sintaxis y Funcionamiento

El bucle do...while es similar al bucle while, pero la condición se evalúa después de que el bloque de código se ha ejecutado, lo que garantiza que el bloque se ejecuta al menos una vez.

Sintaxis: 

do {
  // Código a ejecutar
} while (condición);

2.3.2 Ejemplos de Uso

Ejemplo de un bucle do...while que imprime los números del 1 al 5: 

let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);

2.4 for...in Loop

2.4.1 Sintaxis y Funcionamiento

El bucle for...in se utiliza para iterar sobre todas las propiedades de un objeto enumerable.

Sintaxis: 

for (variable in objeto) {
  // Código a ejecutar
}

2.4.2 Ejemplos de Uso

Ejemplo de un bucle for...in que itera sobre las propiedades de un objeto: 

const persona = {
  nombre: 'Juan',
  edad: 30,
  ciudad: 'Madrid'
};

for (let propiedad in persona) {
  console.log(propiedad + ': ' + persona[propiedad]);
}

2.5 for...of Loop

2.5.1 Sintaxis y Funcionamiento

El bucle for...of se utiliza para iterar sobre los valores de un objeto iterable (como Arrays, Strings, Mapas, Sets, etc.).

Sintaxis: 

for (variable of iterable) {
  // Código a ejecutar
}

2.5.2 Ejemplos de Uso

Ejemplo de un bucle for...of que itera sobre los elementos de un Array: 

const numeros = [1, 2, 3, 4, 5];

for (let numero of numeros) {
  console.log(numero);
}

2.6 switch Statement

2.6.1 Sintaxis y Funcionamiento

La declaración switch evalúa una expresión, comparando el valor de esa expresión con las instancias case, y ejecuta declaraciones asociadas a un caso.

Sintaxis: 

switch (expresion) {
  case valor1:
    // Código a ejecutar
    break;
  case valor2:
    // Código a ejecutar
    break;
  default:
    // Código a ejecutar si no coincide ningún caso
}

2.6.2 Ejemplos de Uso

Ejemplo de una declaración switch que muestra un mensaje dependiendo del día de la semana: 

const dia = 'Lunes';

switch (dia) {
  case 'Lunes':
  case 'Martes':
  case 'Miércoles':
  case 'Jueves':
  case 'Viernes':
    console.log('Día laboral');
    break;
  case 'Sábado':
  case 'Domingo':
    console.log('Fin de semana');
    break;
  default:
    console.log('Día no válido');
}
